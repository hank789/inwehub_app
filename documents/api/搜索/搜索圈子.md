# 搜索标签

## 接口地址

`/api/search/group`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| search_word   | string   | yes      | 搜索关键词 |
| page   | string   | yes      | 页数，默认1 |
