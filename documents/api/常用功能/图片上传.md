# 获取用户信息

## 接口地址

`/api/upload/img`

## 请求方法

```POST ```

## 接口变量

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| img_name  | file   | yes      | 文件名   |

### HTTP Status Code

201

## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": {
    "url": "http://www.test.com/img.png",//图片地址
  }
}
``` 

code请参见[消息对照表](消息对照表.md)
