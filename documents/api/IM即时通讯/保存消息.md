# 保存消息

## 接口地址

`/api/im/message-store`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| text   | string   | yes     | 消息内容 |
| contact_id | integer | no | 聊天的用户id，如果传0，则默认和客服聊天 |


## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": {
  
  }
}
```
