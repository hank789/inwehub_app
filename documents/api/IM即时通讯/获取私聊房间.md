# 获取私聊房间

## 接口地址

`/api/im/getWhisperRoom`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| contact_id | integer | yes | 聊天的用户id |


## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": {
        "room_id": 1,//聊天室id
        "contact_id": 2
  }
}
```
