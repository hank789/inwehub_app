# 发起一个提问

## 接口地址

`/api/question/create`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| description   | string   | yes      | 问题内容 |
| price     | string   | yes      | 提问金额 |
| tags     | string      | yes      | 问题标签 |
| hide | string   | no   | 是否匿名:0匿名,1实名 |

## 返回体

```json5
{
  "status": true,
  "code": 10000,
  "message": "操作成功",
  "data": {
    "price": "188", //提问金额
    "id":"1"  //问题id
    "create_time": "2017-04-18 12:00:05"  //提问时间
  }
}
```
