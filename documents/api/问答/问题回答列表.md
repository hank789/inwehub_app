# 问题回答列表

## 接口地址

`/api/question/answerList`

## 请求方法

```POST ```

## 接口变量

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| question_id  | integer   | yes      | 回答id   |
| page   | string   | yes     | 页数，从1开始 |

### HTTP Status Code

201
