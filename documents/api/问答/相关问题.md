# 相关问题

## 接口地址

`/api/question/relatedQuestion`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| id   | string   | yes      | 问题id |
| limit   | string   | no     | 条数，默认2 |
