# 互动问答-问答列表

## 接口地址

`/api/question/commonList`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| top_id   | string   | no      | 最新id,分页用,默认10条 |
| bottom_id   | string   | no      | 页脚id,分页用,默认10条 |
| tag_id   | string   | no      | 分类id，传0或不传则显示全部 |
