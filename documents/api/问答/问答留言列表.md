# 问答留言列表

## 接口地址

`/api/answer/commentList`

## 请求方法

```POST ```

## 接口变量

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| answer_id  | integer   | yes      | 回答id   |
| page   | string   | yes     | 页数，从1开始 |

### HTTP Status Code

201

## 返回体

    "from": 1,//起始位置

        {
            "id": "1",//回答id
