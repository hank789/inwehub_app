# 问题详情

## 接口地址

`/api/question/info`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| id   | string   | yes      | 问题id |
