# 发起一个提问请求,检查提问条件

## 接口地址

`/api/question/request`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| uuid | string   | sometime   | 用户uuid |
