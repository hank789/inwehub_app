# 发起一个提问请求,检查提问条件

## 接口地址

`/api/question/request`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|


## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": {
    "tags": [
        "系统功能": [
            "SD",
            "MM"
        ],
        "日常操作": [
            "运维"
        ]
    ]
  }
}
``` 


如果返回的`status`是3000,需要跳转到个人信息维护页面
