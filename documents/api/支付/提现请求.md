# 通知列表

## 接口地址

`/api/withdraw/request`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| amount   | string   | yes      | 金额,单位元 |

## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": [
    "withdraw_channel": "wxpay",
    "order_id": "1"
  ]
}
```
