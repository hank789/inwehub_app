# 通知列表

## 接口地址

`/api/pay/request`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| app_id   | string   | yes      | 应用号 |
| amount   | string   | yes      | 金额,单位元 |
| pay_channel   | string   | yes      | 支付方式:alipay,wxpay |
| pay_object_type | string | yes      | 支付对象:ask                 |
