# 文章详情

## 接口地址

`/api/article/detail-by-slug`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| slug   | string   | yes     | 文章slug |
