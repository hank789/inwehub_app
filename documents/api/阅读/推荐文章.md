# 推荐文章到精选

## 接口地址

`/api/article/recommend-app-submission`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| id   | string   | yes     | 文章id |



## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": [
    
  ]
}
```
