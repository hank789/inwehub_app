# 文章回复列表

## 接口地址

`/api/article/comments`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| submission_slug   | string   | yes     | 文章slug |
| sort   | string   | yes     | 排序规则：hot,new |
