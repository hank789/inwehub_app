# 踩回复

## 接口地址

`/api/article/downvote-comment`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| comment_id   | string   | yes     | 回复id |



## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": [
    "type": "cancel_downvote",//类型：cancel_downvote，downvote
  ]
}
```
