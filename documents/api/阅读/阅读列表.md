# 阅读列表

## 接口地址

`/api/article/list`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| page   | string   | yes     | 页数，从1开始 |
| sort   | string   | yes     | 排序规则，hot,new,rising |
| type   | string   | no     | 类型，1为全部，2为文章链接，3为文字，默认1 |
