# 意见反馈

## 接口地址

`/api/system/feedback`

## 请求方法

```POST ```

## 接口变量

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| title  | string   | yes      | 反馈内容,寻找顾问,寻求合作->机遇发布    |
| content  | string   | yes      | 反馈内容    |

### HTTP Status Code

201

## 返回体

```json5
{
  "status": true,
  "code": 0,
  "message": "反馈成功",
  "data": null
}
```

code请参见[消息对照表](消息对照表.md)
