# 动态列表

## 接口地址

`/api/feed/list`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| page   | string   | no      | 页数。默认1 |
| search_type   | integer   | no      | 类型，1:关注,2:全部,3:问答,4:分享,5:他的动态,6:推荐,默认2 |
| uuid   | string   | no      | 用户uuid,search_type为5时必填 |
