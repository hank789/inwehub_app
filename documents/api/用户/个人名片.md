# 用户个人名片

## 接口地址

`/api/profile/resumeInfo`

## 请求方法

```POST ```

## 接口变量

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| uuid | string   | yes      | 用户uuid |


### HTTP Status Code

200
