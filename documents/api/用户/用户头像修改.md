# 修改用户资料

## 接口地址

`/api/profile/postAvatar`

## 请求方法

`POST`

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| user_avatar | file   | yes      | 用户头像 |


###说明


### HTTP Status Code

200

## 返回体
```json5
{
  "status": true,
  "code": 10000,
  "message": "操作成功",
  "data": {
    "user_avatar_id": "1",  //头像id
  }
}
```

code请参见[消息对照表](消息对照表.md)
