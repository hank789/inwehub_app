# 删除教育经历

## 接口地址

`/api/account/edu/destroy`

## 请求方法

`POST`

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| id   | string   | yes      | 主键id |



###说明


### HTTP Status Code

200

```json5
{

  "message": "操作成功",

}
```
