# 删除项目经历

## 接口地址

`/api/account/project/destroy`

## 请求方法

`POST`

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| id   | string   | yes      | 项目经历id |


###说明


### HTTP Status Code

200

## 返回体
```json5
{
  "status": true,
  "code": 10000,
  "message": "操作成功",
  "data": null
}
```

code请参见[消息对照表](消息对照表.md)
