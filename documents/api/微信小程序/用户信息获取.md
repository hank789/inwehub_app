# 获取用户信息

## 接口地址

`/api/weapp/user/info`

## 请求方法

```POST ```

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| code   | string   | yes      | 微信接口提供 |
| iv     | string   | yes      | 微信接口提供 |
| encryptedData     | string    | yes      | 微信接口提供 |


### HTTP Status Code
200
