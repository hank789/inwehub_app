# 加载标签

## 接口地址

`/api/tags/load`

## 请求方法

```POST ```

## 接口变量

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| tag_type  | string   | yes      | 类型:1问题分类,2拒绝回答分类，3行业领域，4产品类型, 5我的擅长(即全部标签)  |
| word      | string   | no      | 搜索关键词            | 
| sort      | string   | no      | 排序规则，0无规则，1热门，默认0            | 
| limit      | string   | no      | 返回条数，不传或传0返回全部            | 


### HTTP Status Code

201
