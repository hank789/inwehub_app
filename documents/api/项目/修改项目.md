# 修改项目信息

## 接口地址

`/api/project/update`

## 请求方法

`POST`

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| id   | string   | yes      | id |
| project_name | string   | yes | 项目名称 |
| project_amount     | string    | yes   | 项目预算 |
| project_address   | string   | yes      | 项目地址 |
| company_name     | string   | yes      | 公司名称 | 
| description     | string   | yes      | 项目简介 |


###说明


### HTTP Status Code

200

## 返回体
```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": {
    "id": "7",
  }
}
```

code请参见[消息对照表](消息对照表.md)
