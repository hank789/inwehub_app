# 用户注册

## 接口地址

`/api/auth/register`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| mobile   | string   | yes      | 注册的手机号码 |
| name     | string   | yes      | 用户名 |
| code     | int      | yes      | 手机验证码 |
| password | string   | yes      | 用户密码 |
| device_code | string   | yes   | 设备号 |

## 返回体

```json5
{
  "status": true,
  "code": 10000,
  "message": "操作成功",
  "data": {
    "token": "bc272b2e87037ded8a5962b33a8cc054", //token
  }
}
```
