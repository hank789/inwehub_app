# 通知列表

## 接口地址

`/api/notifications/list`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| last_id   | string   | no      | 最新id,分页用,默认10条 |

  "code": 10000,
  "message": "操作成功",
