# 关注用户列表
获取当前用户关注用户列表

## 接口地址

`/api/followed/users`

## 请求方法

```POST ```

## 接口变量

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| top_id   | string   | no      | 最新id,分页用,默认10条 |
| bottom_id   | string   | no      | 页脚id,分页用,默认10条 |

### HTTP Status Code

201
