# 关注用户

## 接口地址

`/api/follow/user`

## 请求方法

```POST ```

## 接口变量

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| id  | string   | yes      | 关注的用户id   |

### HTTP Status Code

201

## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": {
    
  }
}
``` 

code请参见[消息对照表](消息对照表.md)
