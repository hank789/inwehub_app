# 关注我的列表
获取关注当前用户的用户列表

## 接口地址

`/api/follow_my/users`

## 请求方法

```POST ```

## 接口变量

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| uuid   | string   | no      | 对应用户uuid，如果不传默认当前用户 |
| page  | int   | yes      | 页数，默认1  |

### HTTP Status Code

201
