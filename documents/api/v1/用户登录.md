# 登录

## 接口地址

```
api/auth/login
````

## 请求方式

`POST`

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| mobile    	| string   | yes   | 登录的用户手机号码 |
| password 		| string   | yes   | 用户密码 |
| device_code   | string   | yes   | 设备号 |

### HTTP Status Code

200

## 返回体

```json5
{
    "token": "bc272b2e87037ded8a5962b33a8cc054", //token
}
``` 

## 返回体字段说明

| name     | type     | description |
|----------|:--------:|:--------:|
| token    | string   | 用户认证token |
