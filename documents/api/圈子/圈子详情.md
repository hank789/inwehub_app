# 圈子详情

## 接口地址

`/api/group/detail`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| id   | integer   | yes     | 圈子id |


## 返回体
