# 圈子分享列表

## 接口地址

`/api/group/submissionList`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| id   | integer   | yes     | 圈子id |
| page   | string   | no      | 页数，默认1 |
| type   | string   | yes      | 显示分类，1全部，2圈主，3精华 |
