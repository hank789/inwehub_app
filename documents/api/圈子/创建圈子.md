# 创建圈子

## 接口地址

`/api/group/store`

## 请求方式

`POST`

### HTTP Status Code

200

## 请求体

| name     | type     | must     | description |
|----------|:--------:|:--------:|:--------:|
| name   | string   | yes     | 圈子名字 |
| description   | string   | yes     | 圈子描述 |
| logo   | base64   | yes     | 圈子封面图 |
| public   | integer   | yes     | 1公开，0私密 |


## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": [
    "id": 1
  ]
}
```
