# 点赞回答
对已点赞的再次请求该接口则为取消点赞

## 接口地址

`/api/support/answer/:id`

## 请求方法

```POST ```

## 接口变量


### HTTP Status Code

201

## 返回体

```json5
{
  "status": true,
  "code": 1000,
  "message": "操作成功",
  "data": {
    "type": "support",//support或unsupport
    "tip": "点赞成功",//或"取消点赞成功"
  }
}
``` 

code请参见[消息对照表](消息对照表.md)
